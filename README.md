# Git Test Project

This project is a sandbox for testing Git workflows and automated releases using [semantic-release](https://github.com/semantic-release/semantic-release).

## Features
- Automated versioning
- Automated changelog generation
- GitHub Releases integration
- Linting and security checks via GitHub Actions

## Scripts
- `npm start`: Runs the development server.
- `npm run semantic-release`: Manually triggers the release process (usually handled by CI).
